# Boss Raid 프로젝트 현황

> 한줄 요약: 로스트아크 스타일 보스전에서 **RL 협동 NPC vs FSM NPC**를 비교하는 새 논문용 환경 구축 및 학습.

---

## 1. 프로젝트 목표

- 원 HCI 논문 (5vs5 대칭 전투, 유저 적응형 NPC) 의 후속 연구
- **비대칭 1보스 4파티 레이드** 환경에서 RL NPC의 협동 능력 평가
- 사용자 평가 (Human-Subject Study) 기반 비교: **RL 조건 vs FSM 조건**

### Seal Break (봉인 해제) 상세

```
[도착 단계] 20턴: NPC 3명이 보스 주변 4장판 중 3개에 각각 이동
  - 딜러에 가장 가까운 장판은 NPC 무시 (딜러용)
  - 보스 정지 (이동/공격/패턴 없음)
  - 보스 무적 (데미지 안 받음 — 기믹 파훼 강제)

[유지 단계] 15턴: 3명 모두 장판 위에서 유지
  - 매 턴 전원 유지 중인지 체크
  - 중간에 누구 하나라도 이탈 → 즉시 전멸
  
[성공] 15턴 유지 완료 → 장판 사라짐 + 보스 3턴 그로기 (딜 타임!)
[실패] 20턴 내 미도착 OR 유지 중 이탈 → 전원 즉사

실시간: 도착 6초 + 유지 4.5초 = 총 10.5초
```

## 6. 학습 구조

### 역할별 독립 네트워크 (v12~)

```
TankNet:    obs(98) → 256 → 256 → Actor(17) / Critic(1)
HealerNet:  (동일 구조, 독립 가중치)
SupportNet: (동일 구조, 독립 가중치)
```

- PPO: lr=3e-4, γ=0.99, GAE λ=0.95, clip=0.2, entropy=0.03
- 각 역할은 자기 경험만 학습 → 역할 간 gradient 간섭 제거

### 보상 설계 — "위험하면 도망, 안전하면 역할" (v21~)

**핵심 원칙: 위험 영역 안에서는 모든 양수 보상 차단 → 보상 충돌 원천 불가**

```
if 위험 영역 안:
    양수 보상 = 0 (역할, 보좌, 근접 전부 OFF)
    위험 패널티 = -5.0/턴
    피격 패널티 = -8.0
    → 무조건 탈출이 최선

if 위험 영역 밖:
    역할 보상 ON (탱커 +2~3.5, 힐러 +힐량×0.5, 서포터 +딜×0.2)
    플레이어 보좌 +1.5 (4m 이내)
    → 역할 수행이 최선
```

| 단계 | 빈도 | 보상 범위 | 목적 |
|---|---|---|---|
| 0단: 플레이어 보좌 | 안전 시 매 턴 | +0.3~1.5 | 딜러 근처 대형 |
| 1단: 역할 수행 | 안전 시 매 턴 | +0.5~3.5 | 탱킹/힐/버프/딜 |
| **2단: 패턴 회피** | **위험 시 매 턴** | **-5.0~-8.0** | **생존 (양수 차단!)** |
| 3단: 기믹 파훼 | 이벤트 | +20~50 | Mark/Stagger/Seal |
| 4단: 보스 처치 | 에피소드 종료 | +1000 / -200 | 최종 목표 |

### 보상 충돌 방지 구조 (v1~v20에서의 교훈)

| 버전 | 문제 | 원인 | 해결 |
|---|---|---|---|
| v1~v4 | 학습 불가 (0%) | 보스 과강 | 커리큘럼 |
| v9~v10 | 역할 미분화 (18%) | 파라미터 공유 | 독립 네트워크 |
| v13 | 힐러 무임승차 | 팀 공유 보상 | free-rider 제거 |
| v14 | 장판 안 피함 (6%) | 역할보상 > 회피보상 | 회피 패널티 강화 |
| v15~v18 | 여전히 장판 위 딜 | 합산 시 양수 | **위험 시 양수 차단** |
| v20 | 보좌 보상이 충돌 | 근접+역할 > 위험 | **위험 시 전체 OFF** |
| **v21** | **충돌 해결** | **이진 모드** | **40% 승률 (450ep)** |

### 커리큘럼 학습

| 단계 | 변경 사항 | 목적 |

| HP 900 시작 | 낮은 난이도 | 승리 경험 확보 |

## 8. 핵심 설계 결정 로그

### 그리드 → 유클리드 전환 (2026-04-14)
- 원 논문 격자 방식에서 연속 공간으로 확장
- 이유: 턴제 느낌 완화, 자연스러운 패턴 회피

### 4방향 → 8방향 이동 (2026-04-15)
- 대각 이동 추가로 패턴 회피 정밀도 대폭 향상
- 행동 공간 13 → 17 (체크포인트 호환 깨짐 → 재학습)

### 파라미터 공유 → 역할별 독립 네트워크 (v12~)
- 공유 방식: 역할 미분화 + 무임승차 문제
- 독립 방식: 역할 분화 명확 + 승률 2.5배 상승

### 보상 "이진 모드" 도입 (v21~)
- v1~v20: 위험 패널티 + 역할 보상 합산 → 위험해도 이득 가능 ("보상 충돌")
- v21: **위험 시 양수 보상 전부 차단** → 충돌 원천 불가 → 학습 속도 급증

### Seal Break 기믹 추가 (v21~)
- 페이즈 전이 시 4장판 등장, NPC 3명이 각각 점유+유지
- **보스 무적 (v22~)**: 기믹 파훼 강제 (딜로 무시 방지)
- 유지 중 이탈 → 즉시 전멸: 높은 협동 요구
- 논문 핵심 기여: "RL이 명시적 규칙 없이 4인 위치 조율 학습 가능한가"

### 이어받기 학습률 교훈 (v17 vs v18)
- v17: lr 3e-4 (높음) → 기존 지식 파괴 → 26%→17% 하락
- v18: lr 5e-5 (낮음) → 기존 지식 보존 → 24%→31% 상승
- **이어받기 시 lr ≤ 5e-5 필수**

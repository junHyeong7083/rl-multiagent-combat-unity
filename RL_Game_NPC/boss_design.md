# 보스 레이드 패턴 설계 문서

로스트아크 스타일 보스 레이드. 각 NPC가 **직업 역할에 맞는 기믹 파훼**를
수행하며 팀워크로 보스를 처치하는 구조.

- **Unity**: 보스 패턴 시각화 + 플레이어 입력 + 사용자 평가
- **Python**: 격자 기반 시뮬레이터에서 NPC 학습 (격자 + Unity Lerp)
- **핵심 비교**: RL NPC vs FSM NPC — 기믹 파훼 협동 품질 차이

---

## 1. 파티 구성

| 슬롯 | 역할 | 조작 | 책임 |
|---|---|---|---|
| **플레이어** | 딜러 (고정) | 직접 조작 | 보스 딜링, 기믹 회피 |
| NPC 1 | **탱커** | RL 또는 FSM | 어그로 유지, 보스 위치 고정, 파티 보호 |
| NPC 2 | **힐러** | RL 또는 FSM | HP 관리, 디버프 해제, 생존 |
| NPC 3 | **서포터** | RL 또는 FSM | 버프/실드, 스태거 기여, 위치 보조 |

4인 파티. 플레이어(딜러) 1명 + NPC 3명.
원 논문 5인 구성에서 레인저를 빼고 보스전에 맞게 축소.

## 2. 보스 스펙 "심연의 군주 (Abyssal Lord)"

| 항목 | 값 | 비고 |
|---|---|---|
| HP | 5,000 | 4인 파티 기준 약 3~4분 전투 |
| 공격력 | 30 (기본) | 패턴별 상이 |
| 이동속도 | 1칸/턴 | |
| 크기 | 2×2 타일 | |
| 어그로 | 누적 데미지 + 도발 가중치 | 탱커가 유지해야 함 |
| 스태거 게이지 | 300 (P2부터) | 파티 전체가 공격해서 깎는 구조 |

### 어그로 시스템

보스는 **어그로 수치가 가장 높은 유닛**을 향해 이동·기본공격한다.

```
어그로[유닛] += 가한 데미지 × 1.0
어그로[유닛] += 도발 사용 시 +200 (탱커 전용)
어그로 전체 × 0.95 매 턴 감쇠
```

탱커가 도발을 제때 쓰지 않으면 딜러(플레이어)에게 어그로가 넘어간다.
이게 **탱커 RL이 학습해야 할 핵심 행동**.

### 스태거 시스템 (P2부터)

특정 패턴 시전 중 스태거 게이지가 표시됨 → 파티 전원이 공격으로 게이지를
깎음 → 0이 되면 보스 2턴 그로기 (딜 타임) / 실패 시 전체 광역 피해.

```
스태거 기여: 기본공격 = 10, 스킬 = 20, 탱커 도발 = 30
제한 시간: 4턴
```

**모든 NPC가 역할과 무관하게 공격에 참여해야** 성공하는 구조.
RL이 "힐러도 잠깐 딜해야 하는 타이밍"을 학습하는지가 관건.

## 3. 페이즈 전이

| 페이즈 | HP 구간 | 핵심 변화 |
|---|---|---|
| **P1 학습기** | 100% → 70% | 기본 패턴 4개. 파티가 역할을 익히는 구간 |
| **P2 격노기** | 70% → 35% | 스태거 체크 추가, 협동 기믹 등장 |
| **P3 최후기** | 35% → 0% | 전멸기 + 역할 강제 배치 기믹 |

페이즈 전이 시 보스 **2턴 무적 + 맵 중앙 귀환 + 전 장판 해제**.
파티 재정비 윈도우.

## 4. 패턴 카탈로그 (8개)

**동시 활성 텔레그래프: 최대 2개.** 안전 타일 0이 되는 상황 방지.

### 패턴 1. 기본 베기 (Slash)
> 어그로 1위 대상 근접 공격. 보스의 기본 리듬.

| 필드 | 값 |
|---|---|
| 등장 | P1~ |
| Telegraph | 2턴 (대상 위 검 아이콘) |
| Shape | 보스 전방 3칸 부채꼴 (격자: 보스 앞 3×1) |
| 피해 | 50 |
| Cooldown | 3턴 |
| **역할별 대응** | |
| 탱커 | 보스 앞에서 맞아야 함 (어그로 유지). 도발 쿨 관리 |
| 힐러 | 베기 직후 탱커 HP 회복 |
| 서포터 | 베기 전 탱커에 방어 버프 |
| 딜러(플레이어) | 보스 후방/측면에서 딜 유지 |

**학습 포인트**: 탱커가 보스 정면을 독점하고, 나머지는 측후방 포지셔닝.

---

### 패턴 2. 돌진 (Charge)
> 랜덤 유닛 1명을 지목, 일직선 돌진.

| 필드 | 값 |
|---|---|
| 등장 | P1~ |
| Telegraph | 3턴 (대상에 빨간 선 + 경로 하이라이트) |
| Shape | 너비 1, 보스→대상 직선 경로 전체 |
| 피해 | 80 |
| Cooldown | 6턴 |
| **역할별 대응** | |
| 지목 대상 | 경로 수직 방향 1칸 이동 (회피) |
| 탱커 | 돌진 종료 지점 근처 이동 → 즉시 도발로 어그로 회수 |
| 힐러 | 맞은 유닛 즉시 힐 |
| 서포터 | 지목 대상에 실드 (못 피할 경우 대비) |

**학습 포인트**: 돌진 후 보스가 파티에서 벗어나므로 **탱커의 어그로 회수 타이밍**이 핵심.

---

### 패턴 3. 바닥 장판 (Ground Eruption)
> 유닛 위치 기반 3지점에 장판 생성. 가장 빈번한 회피 패턴.

| 필드 | 값 |
|---|---|
| 등장 | P1~ |
| Telegraph | 3턴 (3×3 붉은 영역 3개) |
| Shape | 각 3×3, 유닛 현재 위치 ±2 오프셋 |
| 피해 | 60 |
| Cooldown | 5턴 |
| **역할별 대응** | |
| 전원 | 장판 밖으로 이동 |
| 탱커 | 보스 어그로 유지하면서 회피 (보스와 거리 벌어지지 않게) |
| 힐러 | 파티 중앙에 위치 유지 (힐 사거리 확보) |

**학습 포인트**: "회피하되 대형을 유지". 전원이 같은 방향으로 도망가면 다음 장판에 취약.

---

### 패턴 4. 꼬리 휩쓸기 (Tail Swipe)
> 어그로 1위 반대편(보스 후방) 광역 공격. 딜러 견제용.

| 필드 | 값 |
|---|---|
| 등장 | P1~ |
| Telegraph | 2턴 (보스 뒤쪽 부채꼴 하이라이트) |
| Shape | 보스 후방 5×3 (격자: 보스 뒤 5칸 폭 3칸) |
| 피해 | 70 |
| Cooldown | 5턴 |
| **역할별 대응** | |
| 딜러 | 보스 후방이 아닌 **측면**으로 이동 (로아의 백어택 포기 판단) |
| 서포터 | 딜러 근처에 있었다면 같이 이동 |
| 탱커 | 영향 없음 (전방 유지) |

**학습 포인트**: 딜러(플레이어)가 후방에서 딜하고 싶지만, NPC 서포터가
딜러와 같이 측면으로 이동해주는 협동. 로아에서 "백어택 vs 생존" 판단과 동일.

---

### 패턴 5. 집중 표식 (Destruction Mark)
> 비탱커 1명에 표식. 파훼 실패 시 전체 폭발. **핵심 협동 기믹.**

| 필드 | 값 |
|---|---|
| 등장 | P2~ |
| Telegraph | 4턴 (대상 발밑 보라색 원 + 전체 카운트다운) |
| 파훼 조건 | 표식 대상이 아군과 **5칸 이상** 떨어져야 해제 |
| 파훼 실패 | 표식 위치 기준 5×5 폭발, 80 피해 (전원 맞을 수 있음) |
| 파훼 성공 | 표식 해제, 보스 1턴 경직 (딜 찬스) |
| Cooldown | 8턴 |
| **역할별 대응** | |
| 표식 대상 | 파티에서 떨어져야 함 |
| 나머지 전원 | 표식 대상 **반대 방향**으로 이동 |
| 힐러 | 표식 대상이 해제 후 복귀하면 힐 |
| 탱커 | 보스 어그로 유지하면서 표식 대상 반대편 위치 |

**학습 포인트**: RL NPC가 "내가 표식이면 도망, 남이 표식이면 반대로 이동"을
**역할과 무관하게** 학습하는지. FSM으로 이걸 짜면 조건문이 매우 복잡해진다.

### 패턴 6. 스태거 체크 (Stagger Check)
> 보스가 차지 시작. 전원이 제한 시간 내 스태거 게이지를 깎지 않으면 전멸기.

| 필드 | 값 |
|---|---|
| 등장 | P2~ |
| Telegraph | 4턴 (스태거 게이지 바 표시) |
| 게이지 | 300 |
| 기여량 | 기본공격 10, 스킬 20, 도발 30 |
| 성공 | 보스 2턴 그로기 (딜 타임) |
| 실패 | 전체 120 피해 (파티 반 이상 사망 가능) |
| Cooldown | 12턴 |
| **역할별 대응** | |
| 전원 | 보스에 밀착해서 공격 (힐러·서포터도 딜 참여) |
| 탱커 | 도발 사용 (30 기여, 가장 큰 단일 기여) |
| 힐러 | 힐 중단하고 공격 (4턴 동안 딜 우선) |
| 서포터 | 공격 버프 후 딜 참여 |

**학습 포인트**: 힐러가 "지금은 힐보다 딜이 중요하다"를 판단하는 순간.
역할 고정에서 벗어나는 **상황 판단 능력**.
FSM은 "스태거 중이면 공격"으로 하드코딩 가능하지만, RL은 스태거
게이지 잔량에 따라 "안 깎아도 될 것 같으면 힐 유지" 같은 유연한 판단 가능.

### 패턴 7. 전멸기 — 십자 화염 (Cross Inferno)
> 맵을 십자로 가르는 전멸기. **지정된 안전 지대로 이동해야 생존.**

| 필드 | 값 |
|---|---|
| 등장 | P3~ |
| Telegraph | 5턴 (십자 영역 점진적 하이라이트) |
| Shape | 맵 가로 중앙 3줄 + 세로 중앙 3줄 = 십자 |
| 피해 | 300 (즉사급) |
| 안전 지대 | 맵 4사분면 중 2곳 (랜덤, 텔레그래프에 표시) |
| Cooldown | 15턴 |
| **역할별 대응** | |
| 전원 | 표시된 안전 지대로 5턴 안에 이동 |
| 탱커 | 안전 지대 중 딜러 가까운 쪽으로 이동 (보호 우선) |
| 힐러 | 이동 중 저HP 유닛 힐 |
| 서포터 | 이동 느린 유닛에 이동속도 버프 (있을 경우) |

**학습 포인트**: 로아의 "자리 잡기" 기믹. 전원이 **같은 안전 지대로 뭉쳐야**
이후 힐·버프 연계가 가능. RL이 "어느 안전 지대를 선택할지"를 파티 위치
기반으로 **합의**하는지가 핵심. 2곳 중 파티 무게중심에 가까운 쪽을 선택하는
행동이 나오면 성공.

### 패턴 8. 저주의 연결 (Cursed Chain)
> 2명을 연결선으로 묶음. 거리가 3칸 이하로 유지되지 않으면 양쪽에 지속 피해.

| 필드 | 값 |
|---|---|
| 등장 | P3~ |
| Telegraph | 2턴 (두 유닛 사이 사슬 이펙트) |
| 대상 | 랜덤 2명 (탱커-딜러, 힐러-서포터 등) |
| 유지 조건 | 연결된 2명이 거리 3칸 이내 유지 |
| 위반 시 | 매 턴 25 피해 (양쪽 모두) |
| 지속 | 6턴 후 자동 해제 |
| Cooldown | 10턴 |
| **역할별 대응** | |
| 연결된 2명 | 서로 가까이 붙어서 이동 |
| 연결 안 된 유닛 | 연결 쌍을 방해하지 않도록 위치 조절 |
| 특수 상황 | 탱커-딜러 연결 시 → 딜러가 전방으로 와야 함 (위험 감수) |

**학습 포인트**: 로아의 "줄 기믹". 연결 대상 조합에 따라 대응이 완전히 달라짐.
탱커-딜러 연결이면 탱커가 살짝 후퇴 or 딜러가 전방 진입을 **상황에 따라** 판단.
FSM으로 모든 조합을 커버하려면 경우의 수가 폭발한다.

## 6. 보스 FSM 구조

```
[IDLE]
  ├─ HP 임계값 도달? → [PHASE_TRANSITION] (무적 2턴, 중앙 귀환)
  ├─ 쿨타임 가능한 패턴 수집
  ├─ 동시 텔레그래프 < 2개?
  │    ├─ YES → 가중 랜덤 선택 → [TELEGRAPH]
  │    └─ NO  → [BASIC_ATTACK] (어그로 1위 기본 공격)
  └─ [TELEGRAPH] → 카운트다운 → [IMPACT] → [RECOVERY 1턴] → [IDLE]

  * 매 턴: 어그로 테이블 갱신, 어그로 1위 방향으로 1칸 이동
```

## 7. NPC 행동 공간 (원 논문에서 확장)

| ID | 행동 | 사용 역할 |
|---|---|---|
| 0 | 대기 | 전원 |
| 1-4 | 이동 (상/하/좌/우) | 전원 |
| 5 | 보스 공격 (근접) | 전원 |
| 6 | 보스 공격 (스킬) | 전원 |
| 7 | 도발 (어그로 +200) | 탱커 |
| 8 | 방어 자세 (피해 50% 감소, 1턴) | 탱커 |

| 10 | 디버프 해제 (저주 등) | 힐러 |

## 8. NPC 관찰 벡터

원 논문 229차원에서 **보스전 전용으로 재설계**.

| 구성 요소 | 차원 | 설명 |
|---|---|---|
| 자기 상태 | 8 | HP, MP, 위치(x,y), 역할 one-hot(4) |
| 아군 상태 | 24 | 3명 × 8차원 (HP, MP, 위치, 역할) |
| 보스 상태 | 10 | HP비율, 위치(x,y), 페이즈(3), 어그로1위 유닛ID(4) |
| 어그로 | 4 | 각 유닛의 어그로 비율 (정규화) |

## 10. 비교군 FSM NPC

RL과 비교할 FSM. 동일한 관찰 정보를 받는다는 전제.

### 탱커 FSM
```
매 턴:
  if 어그로 1위 아님 AND 도발 쿨 가능 → 도발
  if 보스 텔레그래프 활성 AND 내가 범위 내 → 방어 자세
  if 보스와 거리 > 2 → 보스 방향 이동
  if 스태거 활성 → 보스 공격
  else → 보스 기본 공격

  * 기믹 대응:
  - P5 표식: 내가 표식 → 파티 반대 이동 / 아님 → 무시
  - P7 십자: 가까운 안전 지대 이동
  - P8 연결: 파트너 방향 이동
```

### 힐러 FSM
```
매 턴:
  if 아군 HP < 40% → 치유 (최저HP 우선)
  if 스태거 활성 → 보스 공격
  if 위험 타일 → 안전 이동
  if 파티 평균 거리 > 4 → 중앙으로 이동
  else → 대기

  * 기믹 대응:
  - P5 표식: 내가 표식 → 이탈 / 아님 → 반대 이동
  - P7 십자: 가까운 안전 지대 이동
  - P8 연결: 파트너 방향 이동
```

### 서포터 FSM
```
매 턴:
  if 탱커 HP < 60% AND 실드 쿨 가능 → 방어 버프 (탱커)
  if 그로기 상태 → 공격 버프 (딜러) + 보스 공격
  if 스태거 활성 → 보스 공격
  if 위험 타일 → 안전 이동
  else → 보스 공격 (딜러 근처 위치)

  * 기믹 대응: 위와 동일
```

**FSM 한계가 드러나는 지점**: 기믹 대응은 하드코딩 가능하지만,
"P3 장판 + P8 연결이 동시에 왔을 때 어느 걸 우선할지"같은
**다중 기믹 중첩 상황**에서 우선순위 충돌이 발생. RL은 이걸 경험에서 학습.
